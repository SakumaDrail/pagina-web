Basicos de un PC

Procesador

Unidad funcional de una computadora u otro dispositivo electrónico, que se encarga de la búsqueda, interpretación y ejecución de instrucciones.

Placa Madre

Es una placa de circuito impreso principal de una computadora, que se encarga de dar una conexión a las demás piezas.

Memoria RAM

Es la encargada de ejecutar la mayor parte del software, desde el propio sistema operativo hasta el software de aplicación y otros programas semejantes.

Tarjeta Gráfica

Tiene la capacidad de procesar los datos provenientes del CPU y los transforma en información visual en un dispositivo de salida, como un monitor.

Disco Duro

Es un dispositivo de almacenamiento de datos no volátil, que puede almacenar diferentes archivos o aplicaciones de manera interna.

Fuente de Alimentación

Es el dispositivo encargado de convertir la red eléctrica en una forma de energía adecuada para los componentes de un dispositivo electrónico.
